### Project Highlights

*   **Full-Stack Product Lead**: Led a four-member team to build UIPathFinder, a full-stack AI application that plans students’ daily routes and activities using class schedules, GIS, and weather data.
*   **AI Planning**: Designed the data pipeline and AI planning module integrating LLM reasoning with predictive analytics to recommend personalized itineraries and adaptive campus routes.
*   **Frontend Development**: Developed the frontend with React and Next.js, implementing interactive maps and real-time route updates connected to live occupancy and weather feeds.
*   **Usability Testing**: Conducted usability testing with 40+ students and iterated algorithms and UX flows, improving recommendation accuracy and user experience.
